position:static;

Un elemento con este tipo de posicionamiento simplemente se dispone en el orden normal en el que aparece en la página y ocupa todo el espacio que necesita.

Este elemento tiene posicionamiento estático

position:relative;

Un elemento con position:relative se coloca en principio igual que cualquier elemento estático; de bloque o insertado. Pero entonces sucede algo muy interesante: la caja generada se desplaza según las propiedades top, bottom, left y right.

Lo que hay que recordar sobre el posicionamiento relativo es que sólo se desplaza la caja generada. El elemento sigue estando allí donde estaba en el flujo del documento estático. Aquí es donde "ocupa espacio" respecto a los otros elementos. Eso significa que la caja desplazada puede acabar encima de otras cajas de elementos, ya que éstas siguen actuando como si el elemento con un posicionamiento relativo se hubiera quedado donde debía estar antes de aplicar el posicionamiento. Respecto al flujo del documento, el elemento no se ha movido; es sólo el resultado visual final lo que muestra la caja desplazada.

Este elemento tiene posicionamiento relativo

position:absolute;

Si un elemento posicionado absolutamente no tiene un antepasado posicionado, entonces el bloque contenedor es lo que se llama bloque contenedor inicial, que en la práctica equivale al elemento html.

Pasos para encontrar el bloque contenedor de un elemento con position:absolute;

  1. Mira el elemento padre del elemento posicionado abolutamente. Si éste tiene posicionamiento fixed, absolute o relative haz encontrado el elemento contenedor.
  2. En caso contrario, verifica el elemento padre del padre y empieza de nuevo desde el paso 1 hasta encontrar el elemento contenedor o se haya quedado sin antepasados.
  3. Si llegaste al elemento html, entonces este es el contenedor.

position:fixed;

Los elementos con posicionamiento fijo están fijos en relación con la ventana del navegador en vez de estarlo en relación con el elemento que lo contiene; incluso si se desplaza la página, se mantienen exactamente en la misma posición en la ventana del navegador.

Elemento fijo

position:sticky;

Los elementos con posicionamiento sticky se comportan como elementos con posicionamiento relative hasta que se desplaza fuera de la ventana del navegador, en cuyo caso se comporta como un elemento con posicionamiento fixed.
