HTML y Unicode

Para utilizar un caracter particular de un idioma utiliza el número del código UNICODE correspondiente precedido de "&#" y seguido de ";" (punto y coma), por ejemplo el código 21512, produce ésto: 合

Ejemplos:

Código Caracter Descripción
&#916; Δ Letra griega Delta
&#12354; あ Letra A Hiragana (Japonés)
&#21494; 叶 Idiograma "Hoja" en Chino Simplificado
&#33865; 葉 Idiograma "Hoja" en Chino Tradicional

Pero también puedes mostrar símbolos matemáticos:

8734 = ∞ (Infinito) 8721 = ∑ (Sumatoria) 8736 = ∠ (Ángulo) 188 = ¼ (Fracción)
